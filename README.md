# Marmitaria e Lanchonete - Site Institucional "White"

Este é um site institucional estático desenvolvido para a empresa **Marmitaria e Lanchonete LTDA**, focado em estabelecer credibilidade e autoridade online no nicho de alimentação e refeições caseiras.

## 📋 Informações da Empresa
- **Razão Social:** Marmitaria e Lanchonete LTDA
- **CNPJ:** 65.212.054/0001-66
- **Nicho:** Alimentação / Marmitaria / Lanchonete
- **Localização:** Buíque, PE

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estrutura semântica e otimizada para SEO.
- **CSS3:** Design responsivo (Mobile-first), animações fluidas e tipografia premium.
- **JavaScript:** Interatividade (Menu mobile, scroll suave, efeito de cabeçalho).

## ⚙️ Como Usar
1. Extraia o conteúdo do arquivo ZIP.
2. Abra o arquivo `index.html` em qualquer navegador para visualizar o site.
3. Para fazer o deploy, basta fazer o upload de todos os arquivos para o seu servidor de hospedagem.

## 📝 Notas de Desenvolvimento
- **Links Externos:** Todos os botões e imagens clicáveis estão configurados para redirecionar para `https://google.com`, conforme estratégia de aquecimento de domínio.
- **Responsividade:** O site foi testado em diversos tamanhos de tela para garantir uma experiência perfeita em smartphones, tablets e desktops.
- **Performance:** Imagens otimizadas e ausência de frameworks pesados garantem um carregamento instantâneo.
